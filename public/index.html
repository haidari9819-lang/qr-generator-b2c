FAQ

Ist der QR Code Generator wirklich kostenlos?
+
Ja, vollständig kostenlos. Kein Abo, keine Anmeldung, keine versteckten Kosten. Du kannst unbegrenzt QR Codes erstellen und herunterladen.
Welche QR Code Typen kann ich erstellen?
+
Aktuell unterstützen wir: Website-URLs, WhatsApp-Links, WLAN-Zugangsdaten, digitale Visitenkarten (vCard), PDF, Bild, Video, App-Link und Feedback-Links.
Kann ich mein Logo in den QR Code einbetten?
+
Ja! Im Design-Schritt kannst du ein PNG oder JPG hochladen. Das Logo wird mittig platziert. Der QR Code bleibt dank hoher Fehlerkorrektur scannbar.
In welchen Formaten kann ich den QR Code herunterladen?
+
Aktuell als PNG – direkt aus dem Browser, ohne Server-Upload. Du kannst ihn auch per E-Mail an dich senden lassen.
Werden meine Daten gespeichert?
+
Nein. Der QR Code wird direkt in deinem Browser generiert. Wir speichern keine URLs, Inhalte oder personenbezogenen Daten. DSGVO-konform, Hosting in Deutschland.
